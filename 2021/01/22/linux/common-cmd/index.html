总结工作当中常用的一些 linux 命令行

文件相关

  • 查看文件目录
  • 移动/复制
  • 新建（文件夹/文件）
  • 删除

vim 相关

  • 开启 vim 模式 ,esc 进入命令模式
  • i    insert // 插入
  • e    edit // 修改
  • dd   delete // 删除某一行
  • / ${n||str}    查找行数或查找字符串

三剑客

  • 检测文件变化
  • 查找文件中含有某关键字
  • 获取字符列流

操作符

  • &&:并符，依赖前置条件 cmd A&& cm B ,执行 A 命令后再执行 B 命令
  • |:管道符 A|B ，B 依赖 A 的输出
